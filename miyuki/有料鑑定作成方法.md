# 有料鑑定（本式守護鑑定）作成方法

## 作成の流れ

有料鑑定は以下の2ステップで作成します：

1. **本鑑定構成ファイル**を作成（分析・構成の整理）
2. **本鑑定文**を作成（約5000文字）

---

## ステップ1：本鑑定構成ファイルの作成

### 構成ファイルとは

本鑑定構成ファイルは、**鑑定文を書く前の分析・設計資料**です。

**目的：**
- 四柱推命・紫微斗数の分析結果を整理する
- 無料鑑定で約束した3つの内容を具体化する
- 専門用語を使って裏側の分析を記録する
- 鑑定文の骨組みを作る

**重要：**
- このファイルは相談者には見せない（内部資料）
- 専門用語を自由に使ってOK
- 分析が不足している箇所は[分析必要]と記載

### ファイル名

```
[番号]_[名前]_本鑑定構成.txt
```

例：`056_まゆみ_本鑑定構成.txt`

### 構成ファイルのテンプレート

```markdown
【本式守護鑑定】◯◯様

====================
【基本情報】
====================

■相談者様
生年月日：YYYY年MM月DD日（年齢）
出生時刻：不明 or HH時MM分（時差◯分）
性別：男/女

【四柱推命】
年柱：◯◯（五行）- 通変星 / 十二運
月柱：◯◯（五行）- 通変星 / 十二運
日柱：◯◯（五行）- 本人 / 十二運
時柱：不明 or ◯◯（五行）- 通変星 / 十二運

五行：木◯ 火◯ 土◯ 金◯ 水◯
特殊星：（特殊星のリスト）

【紫微斗数】
命宮：◯◯（命宮）◯〜◯歳
- 主星（◯）
- 副星（◯）
...

年干支：◯◯
時支：◯
陰陽／◯◯局

====================

■相手の方（いる場合）
（同様に命式・命盤を記載）

====================
====================
【本式守護鑑定で分かる3つのこと】
====================
====================

※無料鑑定で約束した3つの内容をここに展開

⛩️1. （タイトル）
━━━━━━━━━━━━━━━━━━━━━━
（無料鑑定で約束した内容）

【四柱推命で見る◯◯】
■◯◯の関係
- 分析内容
- 具体的な解釈

■通変星の組み合わせ
- まゆみさんから見た相手：[分析必要]

- 分析内容

■大運・流年での時期

- 結ばれる最適なタイミング

━━━━━━━━━━━━━━━━━━━━━━
（同様に展開）

⛩️3. （タイトル）
━━━━━━━━━━━━━━━━━━━━━━
（同様に展開）

====================

【結び】
（結びの文章案）
```

### 作成手順

#### 1. 基本情報の整理

命式画像や命盤画像から情報を読み取り、正確に転記する。

**複数人の場合：**
- 相談者
- 相手（恋人、配偶者など）
- 第三者（例：現夫と上司など）

すべての関係者の命式・命盤を記載する。

#### 2. 無料鑑定の確認

無料鑑定で「⛩️本式守護鑑定で分かること」として提示した3つの内容を確認し、
それぞれを詳しく展開する。

#### 3. 四柱推命での分析

各テーマについて：
- 日干同士の関係（相生・相剋）
- 通変星の組み合わせ
- 五行バランスの補完関係
- 十二運の状態
- 大運・流年の流れ
- 特殊星の影響

**プレースホルダーの使い方：**
分析が不足している箇所は`[分析必要]`と記載しておく。

#### 4. 紫微斗数での分析

各テーマについて：
- 命宮の主星
- 関連する宮（夫妻宮、事業宮など）の状態
- 大限・流年の運気
- 化星（化禄・化権・化科・化忌）の影響
- 吉星・凶星の配置

#### 5. 結論の明記

各テーマの最後に：
```
■結論：◯◯
→ [具体的な可能性と必要なアクション]
```

を記載し、分析結果を簡潔にまとめる。

#### 6. プレースホルダーの埋め方

`[分析必要]`と記載した箇所を、命式・命盤を見ながら埋めていく。

**例：**
```
■通変星の組み合わせ
- まゆみさん（丙火）から見た上司（乙木）：偏印
  → 偏印は知恵・学び・庇護を表す
  → 上司はまゆみさんに新しい視点や知恵を与えてくれる存在
  → 精神的な支えとなる相手

- 上司（乙木）から見たまゆみさん（丙火）：傷官
  → 傷官は表現力・才能・魅力を表す
  → まゆみさんは上司にとって魅力的で才能を感じる相手
  → 創造性や感性で惹きつける関係
```

### 注意点

1. **専門用語を使ってOK**
   - このファイルは内部資料なので、四柱推命・紫微斗数の専門用語を自由に使える
   - ただし、後で鑑定文を書くときは、これらを一般向けの表現に翻訳する

2. **分析は正確に**
   - 命式・命盤の読み取りミスがないように注意
   - 五行の数え方、通変星の判定など、基本を間違えない

3. **結論は具体的に**
   - 「可能性が高い」だけでなく、「80%以上」など数値も
   - 「◯月が最適」など具体的な時期も

4. **矛盾がないか確認**
   - 四柱推命と紫微斗数の分析結果が大きく矛盾していないか
   - 無料鑑定で伝えた内容と整合性があるか

### 完成後のチェックリスト

- [ ] すべての命式・命盤情報が正確に転記されているか
- [ ] 無料鑑定で約束した3つの内容がすべて展開されているか
- [ ] [分析必要]のプレースホルダーがすべて埋められているか
- [ ] 各テーマに結論が明記されているか
- [ ] 四柱推命と紫微斗数の両方で分析されているか
- [ ] 具体的な時期（年月）が示されているか

---

## ステップ2：本鑑定文の作成

構成ファイルをベースに、実際の鑑定文を作成します。

**目標文字数：約5000文字**

## 基本方針

### 1. トーンと雰囲気
- **スピリチュアル重視**：情景描写8割、情報2割
- **専門用語は一切使わない**：四柱推命・紫微斗数の用語は裏で分析に使うが、文章には出さない
- **守護神の視点**：守護神が見せてくれる映像、メッセージとして伝える
- **希望と勇気を与える**：基本的には前向きな表現で、ただし現実的な困難も添える

### 2. 文章の流れ

```
1. 冒頭のお礼
2. 結論を先に提示（時期・結果）
   - ポジティブな未来を見せる
   - ただし一筋縄ではいかないことも伝える
3. 無料鑑定で約束した3つの内容を展開
4. 今後半年の流れ（月ごと）
5. オーラの調整方法（具体的な実践方法）
6. パワーストーンの提案
7. 守護神からのメッセージ
8. 結びの言葉
```

## NGルール

### 絶対に使わない表現
- ❌ 「＊」（アスタリスク）を使わない
- ❌ 「━」（罫線）を使わない ※セクションタイトルの⛩️のみ使用可
- ❌ 四柱推命の専門用語（偏印、傷官、劫財、通変星など）
- ❌ 紫微斗数の専門用語（命宮、夫妻宮、化忌星など）
- ❌ 五行（木火土金水）を直接的に説明
- ❌ 断定的なネガティブ表現（「絶対にダメ」「失敗する」など）
- ❌ 冷たい印象を与える表現

### 使うべき表現
- ✅ 「視える」「守護神が見せてくれた」
- ✅ 「魂のレベルで」「運命の流れ」
- ✅ 「エネルギー」「オーラ」
- ✅ 「**強調したい部分は太字で**」
- ✅ 自然や色の比喩（太陽、大地、水、炎、光など）

## 文章構成の詳細

### 1. 冒頭のお礼（テンプレート）

```
【本式守護鑑定】◯◯様

◯◯様、

本式守護鑑定をお受けいただき、
本当にありがとうございます。

あなたの魂が、
本気で変わろうとしている。

その強い意志が、
私のもとへ届きました。

ここから先、
あなたが本当に知るべきことを、
すべてお伝えしていきます。
```

### 2. 結論を先に提示

- **時期を明確に**：「2027年の春」など具体的に
- **良い未来のビジョンを見せる**：映像として描写
- **ただし、困難も伝える**：「簡単ではない」「乗り越える必要がある」
- **行動すべきタイミングを示す**：「2026年9月」など

例：
```
あなたと上司の方が、
手を繋いで笑っている姿が、
はっきりと視えています。

それは、
2027年の春。

桜が咲き始める頃、
二人は新しい人生を歩み始めている。

そこに至るまでの道のりは、
決して平坦ではありません。

でも──
あなたには、それを乗り越える力がある。
```

### 3. 無料鑑定で約束した3つの展開

無料鑑定で「⛩️本式守護鑑定で分かること」として提示した3つを、
それぞれ詳しく展開する。

#### 展開方法：
1. **問いかけから入る**：「こんな経験はありませんでしたか？」
2. **共感する**：「それは〜だからです」
3. **裏にある運命の仕組みを説明**（専門用語なし）
4. **具体的なアドバイス**
5. **時期とタイミング**

#### 相性分析の書き方（専門用語を使わない例）

❌ NG例：
```
あなたは丙火で、相手は乙木です。
木生火の相生関係なので相性が良いです。
```

✅ OK例：
```
あなたが太陽だとすれば、
上司の方は、
あなたを育む大地と水。

あなたが燃やし続けてきた炎を、
彼は優しく支えてくれる。

そして、
あなたが枯れそうになったとき、
新しい命を吹き込んでくれる。
```

### 4. 今後半年の流れ（月ごと）

**前段を入れる**（導入文）：
```
ここから、今後半年間の運気の流れを、
月ごとにお伝えします。
あなたがどう動けば良いか、
具体的に視ていきましょう。
```

**各月ごとの構成**：
- **月のテーマ**：「準備の月」「決断の月」など（必須）
- **その月の運気の特徴**：情景描写で表現
- **具体的にすべきこと**：行動指針
- **注意点やメッセージ**：心構え

**書き方のポイント**：
- 1ヶ月につき3-5行程度
- 短い行で区切り、リズムを作る
- 情景描写を入れる（「嵐の前の静けさ」「水面下では大きな変化が準備されている」など）
- 相談者に語りかける形で

テンプレート：
```
2026年◯月：◯◯の月

（運気の説明：2-3行）

（具体的なアドバイス：2-3行）

（注意点やメッセージ：1-2行）
```

実例：
```
2026年8月：静寂の月

8月は、嵐の前の静けさ。
表面的には、何も変わらない日常が続きます。
でも、水面下では、
大きな変化が準備されています。

この月は、自分を整える時間に使ってください。
オーラを整え、体調を整え、心を落ち着ける。

そして、9月に向けて、
エネルギーを蓄えます。
```

### 5. オーラの調整方法

**必ず前段を入れる**：

なぜオーラを整える必要があるのかを説明してから、具体的な方法に入る。

前段の構成：
1. オーラを整える理由
2. オーラが整うとどんな変化が起こるか
3. 現在のオーラの状態（相談者の状況に合わせて）

前段の例：
```
ここから、あなたのオーラを整え、
本来の力を取り戻す方法をお伝えします。

オーラには3つの種類があります。

あなた自身が持っているオーラ、
相手から受け取るオーラ、
そして環境が与えてくれるオーラ。

自分にとって良いオーラの状態を保つと、
自由でいることができ、
心が穏やかな状態を保てるようになります。

逆に、オーラが整っていないと、
常にイライラし、体力が低下し、
悪いものを引き寄せるようになってしまいます。

（現在のオーラの状態を相談内容に応じて説明）

でも今から、
それを整えていきます。

まずは一つでもいいので、
以下のアクションを行ってください。

これは、あなただけのために
お伝えするアクションです。

（相談者のオーラの種類を明示）
あなたの炎のオーラ、
太陽のような黄金の紅のオーラを
取り戻すために。
```

**具体的な方法を提示**：

- 朝のルーティン
- 色の使い方
- 浄化方法（塩、水、煙など）
- 神社参拝
- アファメーション

**重要：実践しやすいように**
- 「まずは一つでもいいので」という前置きを入れる
- ハードルを下げる
- 具体的で簡単な方法を提示

各方法の例：
```
【朝のルーティン】
毎朝、太陽の光を浴びてください。窓を開け、
太陽の方を向き、両手を広げ、深呼吸を3回。

「私は太陽のように輝く」
「私は愛されている」
「私は守られている」と心の中で唱えます。

【色の力を使う】
赤い小物や金色のアクセサリーを身につけてください。
特に、離婚を切り出す日は、赤いなにかを。

【塩で浄化する】
週に一度、塩を入れた塩風呂に入り、
ネガティブなエネルギーを流します。
(塩の量はほんの少しでも大丈夫です)

【守護神との繋がり方】
月に一度、神社を参拝して、
今の状況と思っている事を
報告してみてください。
```

### 6. パワーストーンの提案

**必ず前段を入れる**：

パワーストーンとは何か、なぜ必要なのかを説明してから、具体的な石の提案に入る。

前段の構成：
1. パワーストーンとは何か
2. なぜ今パワーストーンが必要なのか
3. パワーストーンがもたらす効果（相談内容に応じて）

前段の例：
```
最後に、あなたを支える
パワーストーンをお伝えします。

パワーストーンは、
ただの石ではありません。

長い年月をかけて
地球の中で育まれた石には、
強いエネルギーが宿っています。

そのエネルギーが、
あなたのオーラを安定させ、
運気を高めてくれます。

特に、人生の転換期にある今、
パワーストーンは
あなたの強い味方になります。

（相談内容に応じた具体的な効果）

それらを、
石が静かに後押ししてくれます。
```

**3つの石を提案**（多すぎない）

各石について：
- **効果**：2行程度で簡潔に（説明っぽくならないように）
- **なぜあなたに必要か**：相談者の状況と結びつける（1行程度）

**重要：説明は簡潔に**
- くどくならないように注意
- エッセンスだけを残す
- 各石2-3行以内

石の説明例（簡潔版）：
```
1. ガーネット（柘榴石）
情熱と勇気、新しい恋を引き寄せる。
あなたの炎のオーラと相性抜群です。

2. シトリン（黄水晶）
金運と希望、ネガティブなエネルギーを払う。
黄金のオーラと共鳴します。

3. ローズクォーツ（紅水晶）
自己愛と恋愛運、心の傷を癒す。
傷ついた心を優しく癒します。
```

**ブレスレット提供の形式**：
```
この3つの石を組み合わせたブレスレットを、
私が作成してお送りします。

ご希望の場合は
「詳細希望」と
メッセージをお送りください。

詳細をお伺いし、
あなただけのブレスレットを
お作りいたします。
```

**注意**：
- 自分でブレスレットを作って差し上げる形式
- 「詳細希望」というキーワードで申し込み
- 浄化方法などの細かい説明は不要（提供時に伝える）

### 7. 守護神からのメッセージ

**注意：このセクションは省略可能**

守護神からのメッセージは、嘘くさい印象を与える場合があるため、
必要に応じて**省略しても問題ない**。

省略する場合は、パワーストーンのセクションから
直接結びの言葉に繋げる。

---

**使う場合は、守護神の一人称**で書く。

- 相談者に直接語りかける
- これまでの頑張りを認める
- これから進むべき道を示す
- 「守っている」ことを伝える
- 勇気を与える

テンプレート：
```
「（名前）よ。

お前は、よく耐えた。

（これまでの苦労を認める）

だからこそ、
今、伝える。

（これからすべきこと）

私は、
お前の背中を押す。

（励ましの言葉）

お前の幸せを、
私が守る」
```

### 8. 結びの言葉

- 未来のビジョンをもう一度見せる
- 遠隔守護祈祷を行うことを伝える
- 「幸せになっていい」と許可を与える
- 深雪の署名

## 表現テクニック

### 改行の使い方
- **短い行で区切る**：リズムを作る
- **1〜2行で改行**：読みやすさ重視
- **強調したい言葉の前後**：余白を作る

### 強調の仕方
- 太字：`**重要な時期**` `**結論**`
- ダッシュ：「でも──」「そして──」
- カギ括弧：心の声や具体的な言葉

### 視覚的な演出
```
⛩️ セクションタイトル
```

**注意**：罫線（━━━）は使わない。⛩️マークのみでセクション区切りとする。

### 問いかけと共感
```
まゆみさん、
こんな経験はありませんでしたか？

（具体的な状況を描写）

それは、
（理由を説明）
```

## 分析の進め方（裏側）

### 四柱推命での分析
1. 日干同士の関係（相生・相剋）
2. 通変星の組み合わせ
3. 五行バランスの補完関係

## よく使う表現集

### 時期を表す
- 「◯月は、〜の月です」
- 「この時期が、あなたにとっての『◯◯のとき』です」
- 「運気的に、この時期が最も適しています」

### 未来を見せる
- 「視えています」
- 「守護神が見せてくれました」
- 「はっきりと視えます」

### 共感する
- 「こんな経験はありませんでしたか？」
- 「そう感じていたのではないでしょうか」
- 「心当たりがあるはずです」

### 励ます
- 「大丈夫です」
- 「あなたには、それを乗り越える力がある」
- 「守護神が、あなたを守っています」
- 「もう、〜していいのです」

### つなぎ言葉
- 「でも──」
- 「そして──」
- 「ただし──」
- 「なぜなら──」

### 強調
- 「**（時期や重要な言葉）**」
- 「これは、とても大切なことです」
- 「ここが、最も重要なポイントです」

## 注意事項

1. **長すぎない**：1セクション1000-2000文字程度
2. **具体的に**：抽象的な表現だけでなく、行動を示す
3. **一貫性**：守護神や運気の流れに矛盾がないように
4. **個別性**：テンプレートをベースにしつつ、相談者に合わせる
5. **希望を持たせる**：最終的には前向きな未来を示す

## 文字数の目安

**本鑑定文は約5000文字を目安に作成する。**

### 書き方のポイント

1. **簡潔に、でもエッセンスは残す**
   - 冗長な説明は避ける
   - 重複する表現は削除
   - 情景描写8割、情報2割のバランスを保つ

2. **必須要素**（これらは必ず含める）：
   - 冒頭のお礼
   - 結論（時期・未来のビジョン）
   - 無料鑑定で約束した3つの内容
   - 今後半年の流れ（各月3-5行）
   - オーラ調整方法（前段+具体的方法）
   - パワーストーン（前段+3つの石+ブレスレット提供）
   - 守護神からのメッセージ（省略可）
   - 結びの言葉

3. **文字数調整のコツ**：
   - 各月の流れ：3-5行で簡潔に
   - パワーストーンの説明：各石2-3行
   - オーラ調整方法：各方法2-3行
   - 重要な時期やタイミングは削らない
   - 具体的なアクションは残す
